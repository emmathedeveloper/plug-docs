Widget

In Plug, A widget is a function that takes in an object containing information on how to build an Element.

Widget({ tag: 'button', props:{ className: 'btn-primary', textContent: 'Hello World' }, attributes:{ id: 1 } })

To See this created widget in the DOM, we just have to call the .mount method on the widget with a provided root,

const container = document.querySelector('#app') Widget({ /* configuration */ }).mount({ root: container })

This is Plug at it's most basic form. There are basic widgets which are pre made when you install plug. A few examples are, Text, Button, Container, Anchor and others. We would talk more indepth about widgets in the next chapter.
