Introduction

Installation

Install plug-cli via npm.

$npm install plug-cli

After installing plug-cli, open your terminal in an empty directory and run the command below:

$plug --new MyApp

You can add the --ts flag to the end of the above command if you want a TypeScript project instead.

Install all the dependencies by running the command below:

$npm install

And this would give you a boilerplate plug project. With your terminal still open, run the command below to see your project in the browser:

$npm run dev
