Lifecycle Callbacks

Both The StatefulWidget and StatelessWidget have three major lifecycle callbacks:

StatefulWidget({ onBeforeMount: () => console.log('Widget will soon mount'), onMount: () => console.log('Widget has Mounted'), onDestroy: () => console.log('Widget has been destroyed'), })

onBeforeMount

This callback is ran before the widget is mounted on the DOM. If your widget depends on data from an external source, you can fetch those data in this callback.

onMount

This callback is ran immediately after the widget is mounted on the DOM.

onDestroy

This callback is ran immediately after the widget is removed from the DOM.
